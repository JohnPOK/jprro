CCNA Routing and Switching:

Cisco Certified Network Associate

1. Networking Fundamentals

Lesson Introduction

Basic knowledge of networking types, components, models, protocols, and terminology is necessary if you want to implement and administer Cisco® networks.

Lesson Objectives

In this lesson, you will:

Explain the similarities and differences between network types Explain the role and function of network components Describe network topology architectures Describe the most popular network models Describe protocols that support network communications Compare physical interface and cabling types Describe popular Cisco networking components and the basics of the Cisco IOS commands


2. Configuring Switching

Lesson Introduction

Networks require devices to transfer data among users. The most straightforward network device for LANs is the switch.

Lesson Objectives

In this lesson, you will:

Explain basic concepts of switching Configure basic switch operation Configure VLANs Configure connectivity among multiple switches


3. Configuring IP Addressing

Lesson Introduction

You have configured switches, which work at Layer 2 (and 3) of the OSI Model. Moving up the protocol stack, you need to be able to configure the addressing used by routers before you start configuring the routers themselves.

Lesson Objectives

In this lesson, you will:

Configure IPv4 addresses Configure IPv4 subnets Configure IPv6 addresses Manage network addressing


4. Configuring Routing

Lesson Introduction

Switches and routers are prevalent on practically all networks. While both devices use different addresses, processes, and protocols, they both move data across a network or internetwork. Now that you have configured switches, it should be a bit easier for you to configure routers.

Lesson Objectives

In this lesson, you will:

Describe basic routing concepts Interpret routing tables Configure static routing Configure dynamic routing


5. Configuring Wireless Connectivity

Lesson Introduction

You have spent quite a bit of time dealing with wired networks. You also need t osupport wireless networks and their components.

Lesson Objectives

In this lesson, you will:

Describe the components of wireless LANs Manage WLAN connections Configure clients for WLAN access

6. Configuring IP Network Services

Lesson Introduction

Now that you have configured wired and wireless connectivity, you can start providing services on your network. No matter what job role a person might have, it's fairly certain that they will need access to network resources.

Lesson Objectives

In this lesson, you will:

Describe major services provided on IP networks Configure DHCP Configure NAT Configure NTP Configure DNS on IP networks Use network management functions and protocols


7. Security Fundamentals

Lesson Introduction

A fundamental responsibility of almost every IT technician or administrator is to protect network users, devices, and data from security attacks. This lesson provide you with an introduction to network security on Cisco® networks.

Lesson Objectives

In this lesson, you will:

Describe basic security concepts Manage passwords Configure security for Layer 2 devices Configure security for wireless networking Describe security measures for remote access


8. Automation and Programmability

Lesson Introduction

Today's networks are becoming increasingly more complex to manage and monitor. Technological advancements in automation, controller-based networking, and programming all help to address these complexities so that your network can provide the services that users need.

Lesson Objectives

In this lesson, you will:

Describe the options for automation, device management, and network management Describe controller-based and software defined networking Describe concepts related to programmability and networking


9. Troubleshooting Network Issues

Lesson Introduction

No matter how carefully you plan and manage a network, there will be issues that affect network (and therefore, organizational) performance and operations. Identifying tools and guidelines for troubleshooting network issues can provide yout with a body of knowledge that will prove invaluable to you in the field.

Lesson Objectives

In this lesson, you will:

Describe common processes for troubleshooting network issues Troubleshoot interface and cable issues Troubleshoot switching issues Troubleshoot IPv4 and IPv6 addressing issues Troubleshoot routing issues Troubleshoot WLAN issues Troubleshoot network services issues Troubleshoot network management issues Troubleshoot security issues
